## 🏗️ Project Structure

### Prerequisites

- Node.js >= 20 (LTS recommended)
- Shopify CLI >= 3.0 (must be installed globally)
- Git

### Initial Setup

1. Clone the repository
2. Duplicate `shopify.theme.toml.example` and rename by removing the `example` at the end of the filename
3. update the theme ID, theme password and store password in `shopify.theme.toml`
4. Create a new branch off of `main`
5. Run `shopify theme dev -e dev`
